
## Before start

**Nacos**

```shell
# login nacos/nacos

# nacos windows
startup.cmd -m standalone

# nacos linux/unix/macos
sh startup.sh -m standalone

# or ubuntu
bash startup.sh -m standalone
```

**Redis**

```shell
# redis windows
redis-server.exe

# redis linux/macos
redis-server

# login by redis-client
redis-client
```

**Nginx**

```shell
# nginx windows
start nginx

nginx -s reload
nginx -s quit
nginx -s stop
```
